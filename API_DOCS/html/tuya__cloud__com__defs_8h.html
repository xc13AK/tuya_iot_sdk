固件文件下载回调函数定义

参数
fw[in] 固件文件信息
total_len[in] 固件文件大小（单位字节）
offset[in] 当前回调函数文件下载偏移量
data[in] 当前回调函数已经下载内容
len[in] data的长度
remain_len[out] 通知涂鸦云当前回调函数本次处理内容长度 (*remain_len<=len)，本参数用于解决本地存储时可能出现的字节对齐问题
pri_data[in] 注册回调函数时设置的额外参数
返回
VOID
